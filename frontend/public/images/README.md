# Images Folder

Place your images here:

## Required Images:
- **`logo.png`** - FoodConnect logo (your custom logo with strawberry, heart, fork, and leaf)

### Logo Specifications:
- **Recommended size**: 200x200px or larger (square format works best)
- **File format**: PNG with transparency (supports the glowing effect)
- **Location**: Place your logo file here as `logo.png`

## Image Paths:
All images should be referenced as:
- `/images/logo.png` (for logo - used throughout the site)
- `/images/your-image.jpg` (for other images)
